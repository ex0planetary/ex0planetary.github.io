I develop games and mods for games. Here's some of the stuff I've made over the years.

Duskbringer (2022-Present)

Duskbringer is a standalone game I've been making since I left the Minecraft modding scene in early 2022. It's a metroidvania inspired by games such as Hollow Knight, Ori and the Will of the Wisps, Breath of the Wild, and Minecraft. You play as a lizard exploring a ruined kingdom carved into a giant tree.

It's still early in development and not released yet but I'm quite proud of how it's coming along!

Minecraft Mods (2020-2022)

From 2020-2022 I was cofounder and lead developer of the Minecraft modding team Team Aurora. Over the years we racked up over 17 million total downloads making vanilla-styled mods. I did almost all of the programming for our mods, on top of a decent bit of game design and managing the team.

Enhanced Mushrooms

Enhanced Mushrooms adds a couple new wood types obtained from Minecraft's giant mushrooms, to make them more consistent with the Nether's mushrooms/fungi which have their own wood types. I did all of the game design for this mod, as well as the vast majority of the programming.

Better Badlands

Better Badlands is a small mod that adds a few features to the Badlands biome. It contains a couple of blocks with unique properties, and a new small dark oak tree. A lot of the stuff in this mod was coded by me, although other team members did most of the code for the Thatch block.

Bayou Blues

Bayou Blues adds a new Bayou biome to the game. Apparently it was good enough that one of the most-downloaded mods of all time took inspiration from it and reworked their Bayou biome. Pretty cool I guess. I did all of the code for this one, and actually pumped out a large portion of the mod's code in a single day!

Abundance

Abundance is another biome mod that adds a couple of flower-related biomes. All of the code in the current patch as of March 2022 was written by me.

Fruitful

Fruitful is a little mod that was designed and coded entirely by me. The only part of it that I didn't do was the artwork. It was intended as an experiment to see if I could go from a problem (apples are annoying to farm) to a mod that solved that problem.

Hanami

Hanami was a mod that added several different features based around a cherry blossom forest biome. I didn't do any of the game design for this one but I did all of the code. It was a lot of work and also is the one example I have of me coding entities! Sadly it was abandoned due to some internal strife within the team.

Platform Fighters (2018)

Platform Fighters is a game I developed in 72 hours for Ludum Dare 41. It was made in C++ using the SFML library for graphics. Maybe doesn't quite hold up to the quality of my other projects but I made it in a short timeframe at age 14 so I'm pretty proud of how it turned out.

Super Mario Scralaxy (2018)

Super Mario Scralaxy is a Mario fangame I was making in Scratch alongside some other users of the platform. It was the first game I worked on with a team. I only released one major version but implemented a whole load of new features into it, including a hub world and several new levels.

Blocky World / Blockitto (2017)

Blocky World, later renamed Blockitto, was a game I was developing for the Pokitto DIY console back in 2017. It was inspired by Notch's game Minicraft, and featured the ability to place and remove a variety of different tiles in a 2D top-down world. I eventually abandoned it after running into technical issues getting it to run on hardware, but it still gave me a lot of experience with building a sandbox game.

(Ignore the low framerate in the GIF, I had to remove some frames to get this to upload back in the day! The actual game ran at a smooth 30 FPS.)

Arcade Classics (2017)

Arcade Classics is another game I developed for the Pokitto. This one ended up actually getting released! It's a collection of a bunch of classic arcade-style games such as Pong, Breakout, and Snake. Runs on hardware too which is pretty neat.

Mode 7 Engine (2017)

Not necessarily a game but more of a demo. I developed a 3D rendering engine in Scratch inspired by Mode 7 on the SNES. It could render 3D terrain and place 2D billboard sprites on said terrain. The terrain itself didn't have depth-buffering but the billboard sprites did (although it was a bit buggy). Unfortunately updates to Scratch broke the triangle renderer a bit so it doesn't really work properly anymore.

Yoshi's New Story (2016)

Yoshi's New Story is a Yoshi fangame I made in Scratch at age 12. It features sidescrolling levels, a world map, (mostly) functional collision detection, enemies, etc. Probably the most impressive thing I ever did with Scratch and definitely the most popular.
